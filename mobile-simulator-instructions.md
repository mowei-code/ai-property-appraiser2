# 📱 通用型行動端模擬器 - 使用說明書

這是一款專為網頁開發者設計的輕量級行動端模擬器，支援多種預設設備、自定義尺寸、顏色切換以及跨專案配置遷移功能。

---

## 🛠 核心功能說明

### 1. 裝置切換與自定義
*   **預設設備**：內建 iPhone 13 至 iPhone 17 系列、Pro Max 及 Mini 等主流尺寸。
*   **新增設備**：點擊選單最下方的「新增設備...」，輸入自定義名稱與解析度（寬x高），即可建立專屬設備（如 iPad、Android 手機等）。
*   **管理與刪除**：透過「管理/刪除設備...」視窗，可以查看所有自建設備並進行刪除。

### 2. 環境與外觀自定義
*   **URL 輸入**：在 URL 欄位輸入您的開發地址（如 `http://localhost:5173`），點擊「連線」即可載入。
*   **背景顏色**：使用顏色選擇器更改模擬器後方的背景色，方便在不同視覺環境下測試。
*   **手機外框**：內建「沉穩黑、華麗金、優雅銀、天空藍、珍珠白」五種質感外框顏色。
*   **比例縮放**：支援 50% ~ 100% 的手動縮放，以及「自動 (Auto)」模式，會根據您的瀏覽器視窗自動調整至最佳大小。

### 3. 沈浸模式 (Immersion Mode)
*   點擊右上角的箭頭按鈕，可隱藏工具列與頁尾，進入全螢幕沈浸視圖，專注於 UI 呈現。按 `Esc` 鍵可快速退回普通模式。

### 4. 跨專案遷移 (匯出/匯入)
由於瀏覽器的安全性限制，設備設定儲存在本地端。若需切換電腦或專案：
*   **匯出**：在管理視窗點擊「匯出配置」，系統會將設定轉為 JSON 文字並複製到剪貼簿。
*   **匯入**：在新的模擬器中點擊「匯入配置」並貼上文字，即可瞬間恢復所有自定義設備。

---

## 🚀 快速上手步驟

1.  **啟動**：透過瀏覽器開啟 `mobile-simulator.html`，或使用 `node simulate-mobile.js` 啟動。
2.  **連線**：確認您的開發伺服器（Vite/Webpack）已啟動，並在 URL 欄填入地址。
3.  **調整**：選擇合適的裝置與縮放比例，開始您的 UI 調校工作。

---

## 💡 小提醒
*   本模擬器使用 `localStorage` 儲存您的自定義設定，清除瀏覽器快取可能會導致設定遺失，建議定期「匯出配置」備份。
*   若網頁無法顯示，請檢查 URL 是否包含 `http://` 或 `https://`。
